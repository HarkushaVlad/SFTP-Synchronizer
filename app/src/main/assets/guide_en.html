How to Use the App

This app allows you to synchronize folder states and files between your phone and an SFTP server. It has several key features to help you keep your directories and files up-to-date.

Features

  • Detects new files and replaces older versions on either device (phone or SFTP server).
  • Adds new files from one device to the other.
  • Deletes files to ensure complete directory mirroring between devices.

Setup

To synchronize folders, you need to configure the following settings:

  • Enter SFTP connection details in the settings.
  • Specify the path to the desired directory on both the phone and the SFTP server. Note that directory names must be identical.
  • For files, ensure that file names match between the two devices.

For phone directory selection, you can use a graphical file picker. For SFTP paths, automatic path filling is available if you specify a base directory in the settings.

Synchronization Process

During synchronization, the app performs the following:

  • If a file needs to be replaced due to being older, you will receive a confirmation prompt to approve or cancel the operation, unless auto-confirm is enabled.
  • File deletions will also prompt for confirmation, unless auto-confirm is enabled.
  • New files are copied without requiring additional confirmation, as data loss is not a concern in this case.
Note: Deletion of files can only be detected if the directory has been scanned and its structure logged beforehand. If the structure was not logged, the app will not detect deleted files and may duplicate them if missing on the opposite platform.

Two Files Synchronization Mode

This mode is used when synchronizing exactly two files instead of directories. In this mode, file deletion is not supported. Only file replacement and copying are available.

Additional Features

  • Auto-Confirm: Enable the auto-confirm feature to automatically accept all changes without prompts.
  • Logs: You can view synchronization logs and file operations for troubleshooting purposes. Access logs via the 'Settings' menu.

Getting Started

  1. Configure the SFTP settings in the 'Settings' menu.
  2. Use the 'Choose' button to select the directory or file on your phone.
  3. Enter the SFTP path manually or use the 'Auto-Paste' feature to automatically fill it based on the base SFTP directory path specified in the settings.
  4. Click 'SYNCHRONIZE' to start the synchronization process.
